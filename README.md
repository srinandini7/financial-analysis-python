 Financial Risk Analysis Tool
This project uses Python to calculate the risk and return of a specific stock (TSLA) over a 1-year period.

Key Skills Demonstrated:
Numerical Aptitude: Calculated annualized volatility and cumulative returns.
Python Mastery: Used yfinance for data ingestion and pandas for data cleaning.
Data Visualization: Created visual reports using matplotlib.
